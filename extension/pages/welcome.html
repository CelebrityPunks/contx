Welcome to Explaude

Save tweets from your feed. Use them as context in Claude.

1

Connect to Claude

Paste this command into Claude Code. It installs the bridge that lets Explaude save tweets directly to your machine — no popups, no dialogs, fully automatic.

npx explaude setup --id=loading...
Claude will run this for you. It takes about 5 seconds.
2

Restart Chrome

After setup, close and reopen Chrome so it picks up the new native messaging host.

3

Save any tweet

Right-click on any tweet on Twitter/X and select "Save to Explaude". Tweets auto-save to ~/.explaude/tweets.md and Claude can read them anytime.

Opens Twitter/X in a new tab
