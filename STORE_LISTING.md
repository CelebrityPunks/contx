## Short Description (132 chars max)
Save any tweet with a right-click. Auto-exports as markdown for Claude Code. Your AI gets context from your feed.

## Detailed Description
Explaude lets you save tweets directly from Twitter/X with a single right-click. Every saved tweet is automatically exported as a clean markdown file that Claude Code can read — giving your AI assistant real context from your feed.

HOW IT WORKS
1. Right-click any tweet → "Save to Explaude"
2. Run one setup command in Claude Code
3. Every tweet you save is instantly available to Claude

FEATURES
- One-click tweet saving via right-click context menu
- Auto-exports to ~/.explaude/tweets.md as clean markdown
- Built-in popup to search, manage, and copy saved tweets
- Works on twitter.com and x.com
- Captures tweet text, author, handle, date, URL, and media info
- Handles quote tweets, threads, and media-only tweets
- Dark theme popup matching Twitter/X aesthetic
- 100% local — no data leaves your machine
- Open source: https://github.com/CelebrityPunks/explaude

SETUP (one time)
After installing, paste this into Claude Code:
  npx explaude setup --id=YOUR_EXTENSION_ID
The welcome page shows your exact command. That's it.

BUILT FOR CLAUDE CODE USERS
Explaude bridges your Twitter feed and your AI coding assistant. Save interesting tweets about tools, frameworks, ideas, or threads — then reference them in any Claude session.
